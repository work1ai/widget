  <files>src/chat-store.test.ts, src/markdown.test.ts</files>
  <action>
1. Add gap tests to `src/chat-store.test.ts` in a new `describe('streaming and status')` block. Use the existing `fireClientEvent` helper and `createMockHost` pattern. All new tests must use realistic multi-event sequences (per user decision), not isolated handler calls:

   a. **Token accumulation sequence**: `typing(true) -> token('Hello') -> token(' world') -> message_end` should produce a single finalized agent message with content 'Hello world' and streaming=false. Assert at each step:
      - After typing(true): typingActive=true
      - After first token: typingActive=false, messages length=1, messages[0].streaming=true, messages[0].content='Hello'
      - After second token: messages[0].content='Hello world', still streaming=true
      - After message_end: messages[0].streaming=false, typingActive=false

   b. **Status text lifecycle**: `status('Looking up...') -> token('Found it') -> message_end` should:
      - After status: statusText='Looking up...'
      - After token: statusText='' (cleared by token)
      - After message_end: statusText='' (stays cleared)

   c. **Typing indicator cleared by message_end**: `typing(true) -> message_end` should result in typingActive=false.

   d. **Multiple streaming rounds**: After first complete sequence (token -> message_end), start a second sequence (token -> message_end) and verify messages array has 2 finalized agent messages.

   Each test should start with `store.connect('ws://test', false)` then `fireClientEvent(store, 'connected', { session_id: 'abc' })`.

2. Create `src/markdown.test.ts` with tests for `renderMarkdown`:

   a. **Bold**: `renderMarkdown('**bold**')` contains `<strong>bold</strong>`
   b. **Italic**: `renderMarkdown('*italic*')` contains `<em>italic</em>`
   c. **Links with target="_blank"**: `renderMarkdown('[link](https://example.com)')` contains `target="_blank"` and `rel="noopener noreferrer"`
   d. **Code blocks**: `renderMarkdown('```\ncode\n```')` contains `<pre><code>`
   e. **Inline code**: `renderMarkdown('use `code` here')` contains `<code>code</code>`
   f. **Unordered list**: `renderMarkdown('- item1\n- item2')` contains `<ul>` and `<li>`
   g. **Ordered list**: `renderMarkdown('1. first\n2. second')` contains `<ol>` and `<li>`
   h. **XSS sanitization - script tag**: `renderMarkdown('<script>alert("xss")</script>')` does NOT contain `<script>`
   i. **XSS sanitization - onerror**: `renderMarkdown('<img onerror="alert(1)" src=x>')` does NOT contain `onerror`
   j. **XSS sanitization - iframe**: `renderMarkdown('<iframe src="evil.com"></iframe>')` does NOT contain `<iframe>`

   Import: `import { renderMarkdown } from './markdown.js';`
