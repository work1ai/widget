**src/components/input-area.ts:**

Create a cached TextEncoder at module level: `const encoder = new TextEncoder();`

Define constants:
```typescript
const BYTE_LIMIT = 4096;
const WARNING_THRESHOLD = BYTE_LIMIT - 200; // Show counter when ~200 bytes remaining
```

Export helper function `getByteLength(str: string): number` that returns `encoder.encode(str).length`.

Export interface for input area props:
```typescript
export interface InputAreaProps {
  disabled: boolean;       // from store.inputDisabled
  placeholder: string;     // from widget attribute
  onSend: (content: string) => void;
  onInput: (value: string) => void;  // for mirror sync
  value: string;           // current input value (managed by widget)
  byteCount: number;       // pre-computed byte count
}
```

Export `renderInputArea(props: InputAreaProps): TemplateResult`:
- Renders the auto-growing textarea using CSS grid trick:
  ```html
  <div class="input-grow-wrap" data-replicated-value=${props.value}>
    <textarea
      class="input-textarea"
      placeholder=${props.placeholder}
      ?disabled=${props.disabled}
      .value=${props.value}
      @input=${(e: Event) => props.onInput((e.target as HTMLTextAreaElement).value)}
      @keydown=${(e: KeyboardEvent) => handleKeydown(e, props)}
      rows="1"
      aria-label="Message input"
    ></textarea>
  </div>
  ```
- Send button (inside the input wrapper, positioned absolutely on the right):
  ```html
  <button
    class="send-button ${canSend ? '' : 'send-button--disabled'}"
    ?disabled=${!canSend}
    @click=${() => { if (canSend) props.onSend(props.value.trim()); }}
    aria-label="Send message"
  >
    ${sendIcon}
  </button>
  ```
  Where `canSend = !props.disabled && props.value.trim().length > 0 && props.byteCount <= BYTE_LIMIT`
- Byte counter (only visible near limit):
  ```html
  ${props.byteCount >= WARNING_THRESHOLD ? html`
    <span class="byte-counter ${props.byteCount > BYTE_LIMIT ? 'byte-counter--over' : ''}">
      ${props.byteCount}/${BYTE_LIMIT}
    </span>
  ` : nothing}
  ```

**`handleKeydown` function:**
- If `e.key === 'Enter' && !e.shiftKey`:
  - `e.preventDefault()`
  - If canSend (not disabled, not empty, not over limit): call `props.onSend(props.value.trim())`
- Shift+Enter: do nothing (default textarea behavior inserts newline)
