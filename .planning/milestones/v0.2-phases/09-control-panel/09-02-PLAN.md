Create `tests/playground-controls.test.ts` with vitest. Test strategy:

**Setup helper:**
Create a mock widget element (plain HTMLElement or minimal stub) with:
- `setAttribute` (spy)
- `style.setProperty` and `style.removeProperty` (spy)
- `_wsConstructor` property
- `handleNewConversation` method (spy)
- Ability to dispatch 'w1-connected' and 'w1-disconnected' CustomEvents

Create the controls element, set `controls.widgetEl = mockWidget`, append to document, await `controls.updateComplete`.

**Test groups:**

1. **Theme Colors (CTRL-01):** Find color input for '--w1-accent-color', set its value and dispatch 'input' event. Assert `mockWidget.style.setProperty` called with ('--w1-accent-color', newValue). Test at least 2 color properties.

2. **Theme Position (CTRL-02):** Find 'bottom-left' radio, click it. Assert `mockWidget.setAttribute` called with ('position', 'bottom-left').

3. **Theme Dimensions (CTRL-03):** Find width range input, set value to '400', dispatch 'input'. Assert `mockWidget.setAttribute` called with ('width', '400px'). Same for height.

4. **Bubble Icon (CTRL-04):** Find bubble-icon text input, set value to 'bot', dispatch 'input'. Assert `mockWidget.setAttribute` called with ('bubble-icon', 'bot').

5. **Connection Toggle (CONN-02):** Find mock/real toggle checkbox. Uncheck it (switch to real). Assert `mockWidget._wsConstructor` is undefined and `handleNewConversation` was called. Check it again (switch to mock). Assert `_wsConstructor` is MockWebSocket.

6. **WebSocket URL (CONN-01):** With real mode active, find URL input, set value, dispatch 'input'. Assert `mockWidget.setAttribute` called with ('server-url', url).

7. **Scenario Buttons (MOCK-06):** Set up `MockWebSocket.instance` as a mock with `triggerScenario` spy. Find each scenario button, click it. Assert `triggerScenario` called with correct name ('greeting', 'long-markdown', 'error-protocol', 'error-rejected', 'error-disconnect', 'session-end').

8. **Scenario Disabled State:** Set controls `mockMode = false`, await update. Assert scenario buttons have disabled styling or attribute.

9. **Connection Status:** Dispatch `new CustomEvent('w1-connected')` on mockWidget. Assert `controls.connectionStatus === 'connected'`. Dispatch 'w1-disconnected'. Assert 'disconnected'.

Import `MockWebSocket` from `../playground/mock-ws.ts` and `../playground/controls.ts` (to register the element).

Use `vi.fn()` for spies. Use `el.shadowRoot!.querySelector()` to find controls within shadow DOM. After state changes, `await controls.updateComplete` before asserting DOM.
